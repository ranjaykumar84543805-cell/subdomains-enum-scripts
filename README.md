# Subdomain Enumeration

Bash scripts for subdomain enumeration with **no API keys required**.

## Features

- No API keys needed
- Pure bash scripts
- Multiple enumeration techniques
- Lightweight and fast

## Usage

```bash
chmod +x *.sh
./script.sh domain.com
```

## Methods

- Certificate Transparency logs
- DNS brute force
- Zone transfer attempts
- Common subdomain patterns

## Requirements

- bash
- dig/host
- curl/wget

## Disclaimer

For authorized testing only. Ensure you have permission before scanning any domain.
